Module: InvitePage

Page gérant les invitations à un "souvenir" via un token. Fonctionnalités : - Récupère le token depuis l'URL avec useParams. - Effectue une requête GET à l'API pour valider l'invitation. - Redirige vers la page du souvenir si le token est valide. - Si l'utilisateur n'est pas connecté (401), enregistre le token en localStorage et redirige vers la page de connexion. - Affiche un message d'erreur si le lien est invalide ou expiré.
