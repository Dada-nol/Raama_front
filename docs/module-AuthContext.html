Module: AuthContext

Context gérant l'authentification de l'utilisateur. Fonctionnalités : - Récupère les informations de l'utilisateur connecté à l'initialisation via `/api/user`. - Gère le loading lors de la récupération des données. - Vérifie un `pendingInviteToken` stocké en localStorage et redirige vers le souvenir correspondant si présent. - Permet de se déconnecter via `/api/logout` et de supprimer le token local.
Properties:
Name Type Description
user Object | null Objet utilisateur connecté ou null si non connecté
setUser function Fonction pour mettre à jour l'utilisateur dans le context
loading boolean Indique si les données utilisateur sont en cours de chargement
logout function Fonction pour déconnecter l'utilisateur

import { useAuth } from './AuthContext';
const { user, logout } = useAuth();
